Networking Commands 


ipconfig 

ipconfig diplays all current TCP/IP network configuration values and refreshes DHCP and DNS settings.

Used without parameters, ipconfig displays IPv4 and IPv6 addresses, subnet masks, and default 
gateways for all adapters.


netstat

Netstat displays active TCP connections, ports on which the computer is listening, or established
connections to other computers.


ping 

ping verifies connectivity to a destination by sending Internet Control Message Protocol (ICMP) echo request 
messages to the destination with incrementally increasing time-to-live (TTL) field values. The receipt 
of corresponding echo "Reply" messages are displayed, along with round trip times.


tracert

tracert determines the path taken to a destination by sending an ICMP echo Request to the destination.
The path displayed is the list of routers in the path between the source and destination.
